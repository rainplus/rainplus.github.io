快速上手 Markdown 书写语言

Markdown 是一种标记语言,允许众使用易读的语言写的纯文本格式编写文档,然后转换成有效的 XHTML(或者 HTML).

变体:

  • CommonMark: 2012年提出对 Markdown 进行一个标准化工作.
  • GFM(Github Flavored Markdown,基于 CommonMark)
  • Markdown Extra: 基于 PHP,Python,Ruby 中实现 Markdown. 拓展了 Markdown 以前没有的功能:HTML 块,代码块,表格,定义清单,脚注,缩写…

语法

Markdown 的语法分为几大部分:

  • 标题
  • 段落与换行
  • 引用
  • 列表
    • 无序列表
    • 有序列表
    • 嵌套列表
  • 分隔符< hr>
  • 超链接
  • 图片
  • 强调
  • 字符转义
  • 拓展用法(非原生标准)
    • 删除线
    • 代码块
      • 表格
      • Task List

标题

Markdown 的标题划分为两种方式: 大标题(=========) 和小标题(———) 与六级标题(h1-h6) 进行划分

大标题与小标题

=== 表示h1 大标题 --- 表示h2 小标题

六级标题

由 # 进行声明,# 的个数代表了标题的级别.

# H1
## H2
### H3
#### H4
##### H5
###### H6

展示为:

H1

H2

H3

H4

H5
H6

段落与换行

段落前后是空行或者人为插入< br> 标记符

换行是在行结尾处至少留有 2 个空格符

有序列表,声明方式为 数字.,如下展示与声明一致:

  1. 有序列表
  2. 有序列表
  3. 有序列表

列表嵌套,子列表开头要缩进 2 个空格, 如下展示与声明一致:

  1. 嵌套列表
  • 字列表

分隔符

声明方式为 重叠*或_或-其中一个符号至少三次

***

___

---

超链接

[](http://www.url.com)

图片

![pic alt](path “opt title”)

强调

  1. 使用*内容* 或者 _ 内容_ 表示 斜体
  2. 使用**内容** 或者 __ 内容__ 表示 加粗
  3. 修改的内容中不能与修饰符号之间留空,否则修饰不生效

echo "markdown"
